## Featured Project

### Alarm Walker — Bachelor’s Degree Final Year Project (FYP)

A smart alarm mobile application built with Flutter that helps users wake up more actively by requiring wake-up tasks before dismissing an alarm.

The project includes alarm scheduling, task-based alarm dismissal, wake-up analytics, SQLite local storage, Firebase issue logging, a Flutter Web admin panel, and APK release through GitHub Releases.

**Repository:** [alarm_walker](https://github.com/Pythoenixx/alarm_walker)
