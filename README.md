## 🚀 Featured Project

### Alarm Walker — Degree Final Year Project

Alarm Walker is a smart alarm mobile application developed using Flutter. It helps users wake up more actively by requiring wake-up tasks before dismissing an alarm.

**Repository:** https://github.com/Pythoenixx/alarm_walker

Main features:

* Alarm creation, editing, and deletion
* One-time and repeating alarm support
* Math, retype, shake, and walking tasks
* Wake-up analytics
* SQLite local storage
* Firebase issue logging
* Flutter Web admin panel
* APK release through GitHub Releases
