## 🚀 Featured Project

### Alarm Walker - Degree Final Year Project

Alarm Walker is a smart alarm mobile application developed using Flutter. The app helps users wake up more actively by requiring wake up tasks before dismissing an alarm.

**Repository:** [https://github.com/Pythoenixx/alarm_walker](https://github.com/Pythoenixx/alarm_walker)

Main Features:

* Alarm creation, editing, and deletion
* One-time and repeating alarm support
* Snooze support
* Alarm ringing with sound and vibration
* Math, retype, shake, and walking tasks
* Wake-up analytics
* SQLite local storage
* Firebase issue logging
* English and Malay interface support
* Flutter Web admin panel
* APK release through GitHub Releases
